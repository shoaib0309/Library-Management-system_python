# Library-Management-system_python

📚 Library Management System (Python & OOP)
A Python-based application that simulates a digital library environment. This project focuses on Object-Oriented Programming (OOP) to manage a collection of books, allowing users to perform basic operations like adding, viewing, and searching for books.

🎯 Project Features
Add Books: Register new books into the system with their title and author details.

View Collection: Display a list of all books currently available in the library.

Smart Search: Search for specific books by title with case-insensitive matching.

Availability Tracking: Each book object maintains its own status (available/not available).

🛠️ Programming Concepts Used
Classes & Objects: Created a Book class to define book properties and a Library class to manage the collection.

List Management: Used Python lists to store and iterate through book objects.

Encapsulation: Grouped data (title, author) and methods (display_info) together for better code structure.

Control Flow: Implemented a while loop and conditional logic to create an interactive command-line interface.
